# DABench Task 0 - Calculate Mean Fare

This task asks participants to compute the mean ticket fare from the provided Titanic dataset.

### Submission Format
- Provide a single row with `id` and `answer` columns.
- The answer must use the template `@mean_fare[value]`, where `value` is the rounded mean fare.

### Evaluation
- Submissions are evaluated using an exact-match checker with a small tolerance on the floating value.
